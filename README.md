# 📚 Учёт успеваемости студентов АВПК

Веб-приложение для автоматизации учёта успеваемости студентов **Актюбинского высшего политехнического колледжа**.

Дипломный проект. Работает полностью в браузере без серверной части — данные хранятся в облаке Firebase.

## ✨ Возможности

- 🔐 **Регистрация и авторизация** через Firebase Authentication (email + пароль)
- 👥 **Управление студентами и группами** — добавление, удаление, фильтрация
- 📚 **Учёт предметов и преподавателей**
- 📝 **Выставление оценок** по 100-балльной шкале (10–100) с буквенным эквивалентом A/B/C/D/F
- 📊 **Отчёты и графики** — успеваемость по группам, по предметам, рейтинг студентов
- 🎭 **Три роли пользователей** — администратор / преподаватель / студент
- 📱 **Адаптивный интерфейс** — работает на компьютере, планшете, смартфоне

## 🛠 Технологии

- **HTML5** — структура страниц
- **CSS3** — стилизация (Grid, Flexbox, адаптивный дизайн)
- **JavaScript (ES6 modules)** — логика приложения
- **Firebase** — облачная база данных и авторизация
  - Firestore — хранение данных
  - Authentication — регистрация пользователей
- **Chart.js** — построение графиков

## 📊 Шкала оценок

| Балл | Буква | Оценка |
|------|:-----:|--------|
| 90–100 | **A** | Отлично |
| 75–89  | **B** | Хорошо |

## 🚀 Как запустить проект

### Шаг 1. Создать проект в Firebase

1. Перейти на [https://console.firebase.google.com/](https://console.firebase.google.com/)
2. Нажать **«Add project»** (Создать проект)
3. Ввести название (например, `avpk-grades`), нажать «Continue»
4. На шаге Google Analytics — можно отключить и нажать «Create project»
5. Дождаться создания проекта

### Шаг 2. Включить Authentication

1. В левом меню → **Build → Authentication**
2. Нажать **«Get started»**
3. Перейти на вкладку **«Sign-in method»**
4. Выбрать **«Email/Password»** → включить переключатель **«Enable»** → **Save**

### Шаг 3. Включить Firestore Database

1. В левом меню → **Build → Firestore Database**
2. Нажать **«Create database»**
3. Выбрать **«Start in test mode»** (для разработки) → **Next**
4. Выбрать локацию (например, `eur3` или `europe-west`) → **Enable**

### Шаг 4. Зарегистрировать веб-приложение

1. В Project Overview (главная страница проекта) нажать на значок **`</>`** (Add web app)
2. Ввести название приложения (например, `avpk-web`)
3. **НЕ ставить** галочку «Firebase Hosting» (не нужно)
4. Нажать **«Register app»**
5. Скопировать содержимое объекта `firebaseConfig` — выглядит так:

```javascript
const firebaseConfig = {
    apiKey: "AIzaSy...",
    authDomain: "avpk-grades.firebaseapp.com",
    projectId: "avpk-grades",
    storageBucket: "avpk-grades.appspot.com",
    messagingSenderId: "123456789012",
    appId: "1:123456789012:web:abc..."
};
```

### Шаг 5. Подставить ключи в проект

1. Открыть файл `src/js/firebase-config.js` в любом редакторе (VS Code, Notepad++, Блокнот)
2. Найти блок `firebaseConfig`
3. Заменить значения на скопированные из Firebase Console
4. Сохранить файл

### Шаг 6. Запустить локально

⚠️ Просто открыть `index.html` двойным кликом **НЕ получится** — Firebase требует HTTP-сервер.

**Вариант А: VS Code + Live Server** (самый простой)
1. Установить расширение **«Live Server»** в VS Code
2. Открыть папку `src/` в VS Code
3. Правой кнопкой на `index.html` → **«Open with Live Server»**
4. Откроется браузер с адресом `http://127.0.0.1:5500`

**Вариант Б: Python**
```bash
cd src
python -m http.server 8000
```
Открыть `http://localhost:8000`

**Вариант В: Node.js (npx)**
```bash
cd src
npx serve
```

### Шаг 7. Начать пользоваться

1. Открыть приложение в браузере
2. Перейти на вкладку **«Регистрация»**
3. Зарегистрироваться как **Администратор** (первый пользователь)
4. Войти в систему
5. Создавать группы, студентов, предметы, выставлять оценки

## 🌐 Деплой на GitHub Pages

### Шаг 1. Создать репозиторий на GitHub

1. Зайти на [https://github.com/](https://github.com/) → New repository
2. Имя: `avpk-grades` (или любое другое)
3. Public → Create repository

### Шаг 2. Залить проект

В терминале (PowerShell / cmd):

```bash
cd путь/к/папке/diploma_project
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/ВАШ_ЛОГИН/avpk-grades.git
git push -u origin main
```

### Шаг 3. Включить GitHub Pages

1. В репозитории → **Settings** → **Pages**
2. **Source:** Deploy from a branch
3. **Branch:** `main`, **Folder:** `/src`
4. **Save**

Через 1–2 минуты сайт будет доступен по адресу:
```
https://ВАШ_ЛОГИН.github.io/avpk-grades/
```

### Шаг 4. Добавить домен в Firebase

⚠️ Чтобы Firebase Auth работал на GitHub Pages:

1. Firebase Console → Authentication → **Settings** → **Authorized domains**
2. Нажать **«Add domain»**
3. Ввести: `ВАШ_ЛОГИН.github.io`
4. **Add**

Теперь регистрация и вход будут работать с GitHub Pages.

## 🔒 Правила безопасности Firestore (опционально)

По умолчанию БД в тестовом режиме открыта для всех. Через 30 дней доступ закроется.

Чтобы настроить правила безопасности, в Firebase Console:
**Firestore Database → Rules → Edit rules**

Вставить:

```javascript
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    // Только авторизованные пользователи могут читать/писать
    match /{document=**} {
      allow read, write: if request.auth != null;
    }
  }
}
```

Нажать **Publish**.

## 👤 Роли пользователей

| Роль | Возможности |
|------|------------|
